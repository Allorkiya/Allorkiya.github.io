A.小红的顺子构造

题目描述

链接：https://ac.nowcoder.com/acm/contest/126636/A

小红定义 \(k\) 张牌是一个「顺子」，当且仅当这些牌从小到大排序后，满足对于 \(1 \sim k - 1\) 的每个 \(i\)，都有 \(a_i+1=a_{i+1}\)。例如，\(\{4,3,6,7,5\}\) 能构成一幅顺子。 现在小红想找到一个由 \(5\) 张牌组成的，包含 \(x\) 这一数字的「顺子」，请你帮帮她。我们可以证明，答案一定存在。

数据范围

  • \(1 \leq x \leq 10\)

思路

错了一发，sb了一下一定要注意。默认构造以k为开头的长度为5的顺子，如果某个数超过13，计算超过13的数字数量，再向左拓展输出即可。、

C.小红的好数组构造

链接：https://ac.nowcoder.com/acm/contest/126636/C

题目描述

小红定义一个数组是好数组，当且仅当数组中每个数都出现了偶数次，且所有元素均为正整数。特殊的，\(\{\}\)（空数组） 也是一个好数组。

现在小红想要找到一个长为 \(n\) 的数组，使得：

  • 对于任意整数 \(d \left(0 \leqq d < k\right)\)，删除数组中任意 \(d\) 个下标后，得到的数组都不是好数组；

  • 一种删除 \(k\) 个下标的方式，使得得到的数组是好数组。

    请你帮帮她。

数据范围

  • \(1 \leq n \leq 2 \times 10^5\)
  • \(0 \leq k \leq n\)

思路

题目等价于只需要构造一个长度为n的数组，是的奇数频次的值的个数等于k即可。

可行性条件：

n是所有频次之和，奇数频次的有k个，则总和的奇偶性满足：

  • 奇数个奇数之和与k同奇偶，偶数频次贡献为偶数
  • 所以n和k必须同奇偶，且\(k \leq n\)

所以：

  • 先放1,2,3…k各一次，这k个值频次都是1，奇数。
  • 剩余n-k个位置用成对的1填充，每增加两个1不改奇偶性，仍为奇数。
  • 这样最终奇数频次的值正好是1,2,3…k共k个

复杂度：O(n)

D.小红的左看右看构造

题目描述

链接：https://ac.nowcoder.com/acm/contest/126636/D

在此题中，我们认为数组以从左到右的顺序排列。

对于一个数组 \(a\)，小红定义「左看」与「右看」为：

  • 「左看」：从左到右依次记录数组中满足“大于其左侧所有数”的数组成的新数组（按取出顺序）。
  • 「右看」：从右到左依次记录数组中满足“大于其右侧所有数”的数组成的新数组（按取出顺序）。

如，设一个数组 \(b = \{2,3,4,3,5,1\}\)，则「左看」为 \(\{2,3,4,5\}\)，「右看」为 \(\{1,5\}\)。

小红给出了一个长为 \(x\) 的数组 \(c\) 与一个长为 \(y\) 的数组 \(d\)，他希望小苯能构造一个长为 \(n\) 的数组，使得 \(c\) 是其「左看」，\(d\) 是其「右看」。请你帮帮小苯。

特殊的，保证数组 \(c\)，\(d\) 严格单调递增。

数据范围

  • \(1 \leq x, y, n \leq 2 \times 10^5\)
  • \(1 \leq c_i, d_i \leq 2 \times 10^5\)

思路

数组一定是按数组最大值分开的，是c和d数组的最后一个值

构造方法：令mxv为数组最大值

  • 当y > 1
    • 把c[0..x-2]放在数组最左边依次放好
    • mxv放在位置x
    • d[0..y-2]放在数组最右边，从后往前放
    • 其他没填的位置都放1
  • 当y = 1
    • 此时d = mxv
    • 把c[0..x-2]放在最左边
    • 把mxv放在a[n]
    • 其他填1
    • 若x=1，还要保证a[1] = mxv（可以让首位都是mxv，不影响左右看）

边界条件：

  • \(x, y \leq n\)
  • \(c[x] = d[y]\)
  • \(n \geq x + y - 1\)

时间复杂度：O(n)

E.小红的完全平方数构造

题目描述

链接：https://ac.nowcoder.com/acm/contest/126636/E

小红拿到了一个整数 \(n\)，她想要在这个整数的十进制表示末尾拼接一个非空的数字串（如将 \(\underline{123}\) 修改为 \(\underline{123}0987654\)），使得修改过后的新数 \(n'\) 是一个完全平方数，请你帮帮她。

数据范围

  • \(1 \leq T \leq 10^5\)
  • \(1 \leq n \leq 10^8\)

思路

直接遍历后面添加了几位，对于添加了k位的情况，数字的范围为\([n \times 10^k, (n + 1) \times 10^k - 1]\)则满足题意。

判断区间内是否有完全平方数：

  • 令\(a = \lceil L \rceil,b = \lfloor R \rfloor\)
  • 若\(a \leq b\)，则\(a^2 \in [L, R]\),输出即可

时间复杂度\(O(Tk)\)

F.小红的基环树染色构造

题目描述

链接：https://ac.nowcoder.com/acm/contest/126636/F

小红拿到了一棵由 \(n\) 个点组成的基环树，她希望将每个节点都染成红，黄，蓝三种颜色中的一种，使得任意两个相邻节点的颜色都不相同。请你帮帮她。我们可以证明，答案一定存在。

数据范围

  • \(4 \leq n \leq 2\times 10^5\)
  • \(1 \leq u_i,v_i \leq n\)

思路

先找环，可以用拓扑排序找到环上的点有哪些。

先将环进行染色：如果环长是偶数，1,2,1,2…交替染即可；如果是奇数，前n-1个用1,2,1,2…交替染色，最后一个染3。

对于每个点衍生的环外分支，先确定色集，比如如果环上的点色是1，连接的那条链的色就是{2,3}

时间复杂度：O(n)

B/G.小红的字符串构造（easy/hard）

题目描述

链接：https://ac.nowcoder.com/acm/contest/126636/G

本题为问题的困难版本，两题的唯一区别在本题不保证给出数组中的「01串」两两有前缀关系。

我们称一个字符串为「01串」，当且仅当其中仅含有字符 \(\texttt{0}\) 和 \(\texttt{1}\)。

小红拿到了长为 \(n\) 的「01串」数组 \(s\)。现在她想要构造一个字符串 \(s '\)，满足 \(s\) 中恰好有 \(k\) 个「01串」（可以相同）是 \(s'\) 的前缀。请你帮帮她

【名词解释】

前缀：从字符串开头到任意位置的子串。如对于字符串 \(\texttt{1234}\)，他的前缀有 \(\texttt{1},\texttt{12},\texttt{123},\texttt{1234}\) 和空串。

数据范围

  • \(1 \leq k \leq n \leq 2 \times 10^5\)
  • \(1 \leq |s_i| \leq 2 \times 10^5\)
  • \(\sum |s_i| \leq 2 \times 10^5\)

思路

对每个字符串插入trie树，每个终止节点记一个权值，这样问题就等价于在trie树上找一段路径，使得路径经过的点的权值为k，输出对应路径的01串即可。

时间复杂度\(O(\sum |s_i| + Trie节点数)\)
